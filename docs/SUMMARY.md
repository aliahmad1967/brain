# Reading Order

For a new reader or AI agent onboarding to the project, consume documents in this order:

| Step | Document | ID | Purpose |
|------|----------|----|---------|
| 1 | Vision | BRN-SPEC-001 | Why this project exists |
| 2 | Mission | BRN-SPEC-002 | What we aim to achieve |
| 3 | Principles | BRN-SPEC-003 | Core beliefs guiding decisions |
| 4 | NonGoals | BRN-SPEC-005 | What is explicitly out of scope |
| 5 | Goals | BRN-SPEC-004 | Measurable objectives |
| 6 | PRD | BRN-SPEC-006 | Full product requirements |
| 7 | Personas | BRN-SPEC-008 | Who we are building for |
| 8 | User Stories | BRN-SPEC-007 | Feature-level narratives |
| 9 | SAD | BRN-SPEC-009 | Software architecture |
| 10 | AI Architecture | BRN-SPEC-010 | AI subsystem design |
| 11 | Storage Architecture | BRN-SPEC-011 | Data layer design |
| 12 | API Specification | BRN-SPEC-012 | API contracts |
| 13 | ADRs | BRN-DEC-* | Key decisions and trade-offs |
| 14 | Engineering Guide | BRN-GUIDE-001 | Development workflow |
| 15 | Guides (Desktop/Web/Testing) | BRN-GUIDE-* | Implementation guides |
| 16 | Roadmap | BRN-REF-001 | Future direction |
